# Batch常用命令

## 文件处理

### 显示目录

```batch
dir
```


### 删除文件

```batch
del
```


## 编程

### 设置变量

```batch
set
```


### 设置系统变量

```batch
setx
```


### 循环

```batch
for
```


## 其它

### 在屏幕上打印字符串

echo

#### 关闭回显

```batch
@echo off
```

#### 开启回显

```batch
@echo on
```

### 清除屏幕

```batch
cls
```

### 注释

```batch
rem
```

```batch
::
```
