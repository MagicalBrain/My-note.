## 导入其他文本文件到markdown

使用`@import`命令，例如 导入一个`.cpp`文件里的代码到markdown里

```bash
@import "../../..//C++/C++11/code/find.cpp"
```

这里输入的是相对路径。

@import "../../..//C++/C++11/code/find.cpp"
