#### 类的静态成员

在类的成员声明前面加static
```cpp
class MyType
{
    static int num;
};
```

静态成员是独立于类的实例化对象之外的，即不与类对象绑定使用。
所有的类对象共享类的静态成员。

#### 类的静态成员函数

不与类对象绑定使用。
不可定义为const
不可在函数体内使用this指针
