### 问题7
**可控位数的四舍六入的java的实现**
主要思路：

```java
public static double sslr(double x,int a)
    {
        int n,m1,m2 ;

        switch (a)
        {
            case 1:
                n = 10;
                break;
            case 2:
                n = 100;
                break;
            case 3:
                n = 1000;
                break;
            default:
                n = 10;
                break;
        }

        m1 = (int) (x*n*10) % 10;
        m2 = (int) (x*n) % 10;
        if (m1 == 5)
        {
            if (m2 % 2 == 0)
                x = x -  1/(n*10.0);
        }

        switch (a)
        {
            case 1:
                x = Double.parseDouble(String.format("%.1f",x));
                break;
            case 2:
                x =  Double.parseDouble(String.format("%.2f",x));
                break;
            case 3:
                x = Double.parseDouble(String.format("%.3f",x));
                break;
            default:
                break;
        }

        return x;
    }
```
