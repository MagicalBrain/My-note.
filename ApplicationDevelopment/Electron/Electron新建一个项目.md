# Electron新建一个项目

## 新建一个文件夹

```bash
mkdir -p [the path your want to start]
cd [the path your want to start]
```

## 运行npm init命令

```bash
npm init
```

## 下载安装Electron模板

```bash
npm install electron --save-dev
```
