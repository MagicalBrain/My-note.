# Git如何设置默认文本编辑器

```bash
git config --global core.editor [your editor name]
```

* Vim
```bash
git config --global core.editor vim
```

* Nano
```bash
git config --global core.editor nano
```

* vscode
```bash
# 注意，Linux下vscode叫code，所以这里我不确定是不是vscode，可能是code
git config --global core.editor vscode
```
