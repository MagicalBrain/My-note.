# C++ Lambda表达式

关于lambda的基础知识可以参考《C++ Primer 5th》的10.3.2部分以及我自己相关的笔记

## lambda的组成（结构）

一个完整的lambda表达式组成如下：

```cpp
[ capture-list ] ( params ) mutable(optional) exception(optional) attribute(optional) -> ret(optional) { body } 

```
