# 00深拷贝和浅拷贝

在C++里普通类型的复制是很简单的：

```cpp
int a = 10;
int b = a;
```

而类对象和普通变量不同，内部结构比较复杂，存在各种成员变量。
举个类对象拷贝的例子：

```cpp
#include <iostream>
using namespace std;

class CExample {
private:
    　int a;
public:
    　CExample(int b)
    　{ a=b;}
    　void Show ()
    　{
        cout<<a<<endl;
    }
};

int main()
{
    　CExample A(100);
    　CExample B=A;
    　B.Show ();
    　return 0;
} 
```
