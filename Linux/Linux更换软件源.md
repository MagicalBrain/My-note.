# Linux更换软件源

## 方法
其实就是编辑```/etc/apt/sources.list```文件

编辑前记得备份

编辑后运行：

```bash
sudo apt-get update
sudo apt-get upgrade
```


以下源在切换的时候注意对应版本

## 软件源

在```/etc/apt/sources.txt```文件的最前面根据Linux的版本添加对应的内容。
