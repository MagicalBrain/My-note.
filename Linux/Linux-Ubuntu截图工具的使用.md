# Linux-Ubuntu截图工具的使用

## 全屏截图

PrintScreen
PrtSc

## 快速截取当前窗口

Alt + PrintScreen

## 截取任意矩形的内容

Shift + PrintScreen

## gnome-screenshot截图工具的使用

按win键，或点击左侧启动栏顶端的 Ubuntu 按钮，调出 DASH；在搜索框里输入 screen 或 截图，单击那个相机图标，就可以启动截图工具主窗口了。

也可以直接在命令行里输入`gnome-screenshot`
